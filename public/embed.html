Chatbot Embed Example

This page demonstrates how to embed the chatbot on any website. The chatbot starts as a small icon, and can be maximized to appear in the center of the screen.

How to Embed

Copy and paste the following code into your website:

<script>
window.addEventListener('message', function(event) {
  if (event.data && event.data.type === 'chatbot-resize') {
    const iframe = document.getElementById('chatbot-iframe');
    if (iframe) {
      if (event.data.centered) {
        // Centered maximized view
        iframe.style.width = event.data.width + 'px';
        iframe.style.height = event.data.height + 'px';
        iframe.style.position = 'fixed';
        iframe.style.left = '50%';
        iframe.style.top = '50%';
        iframe.style.transform = 'translate(-50%, -50%)';
        iframe.style.bottom = 'auto';
        iframe.style.right = 'auto';
        iframe.style.borderRadius = '16px';
        iframe.style.zIndex = '10000';
      } else {
        // Normal corner view
        iframe.style.width = event.data.width + 'px';
        iframe.style.height = event.data.height + 'px';
        iframe.style.position = 'fixed';
        iframe.style.bottom = '24px';
        iframe.style.right = '24px';
        iframe.style.left = 'auto';
        iframe.style.top = 'auto';
        iframe.style.transform = 'none';
        
        // Set border radius based on size
        if (event.data.width <= 100 && event.data.height <= 100) {
          iframe.style.borderRadius = '50%'; // Circular for icon mode
        } else {
          iframe.style.borderRadius = '16px'; // Rounded corners for chat mode
        }
      }
    }
  }
});
</script>

<iframe
  id="chatbot-iframe"
  src="https://ct-chatbot-node.onrender.com/"
  style="position: fixed; bottom: 24px; right: 24px; width: 71px; height: 71px; border: none; border-radius: 50%; box-shadow: 0 4px 24px rgba(0,0,0,0.18); z-index: 9999; background: #fff; transition: all 0.3s ease;"
  allow="clipboard-write"
  title="Chatbot"
></iframe>

Features

  • Starts as a small circular icon (71x71px)
  • When opened, it shows in normal state
  • Can be maximized to center of screen (50% of viewport width)
  • Smooth transitions between states
